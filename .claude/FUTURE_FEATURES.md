# FUTURE_FEATURES.md — 将来実装予定機能リスト

> MVP完成後に順次実装していく機能のバックログ。
> アイデアを忘れないために残しておく。

---

## 🐾 ペット育成

- [ ] **レベルアップシステム**
  - 経験値テーブル: `Math.floor(100 * Math.pow(level, 1.5))`
  - レベルアップ時にスキル解放・ドロップ率UP
  - レベル上限: 100

- [ ] **食事による外見変化**
  - 食べ物によって色が変わる
  - 食べ物によってサイズが変わる（小さくなったり大きくなったり）
  - 発光エフェクトが付くものもある
  - 効果に持続時間あり（永続のものも）

- [ ] **種族システム**
  - 種族候補: ドラゴン・ユニコーン・スライム・フェニックス・ゴーレム
  - 種族によってドロップ品が違う（鱗・角粉・体液など）
  - 種族によって使えるスキルが違う

## ⚡ スキルシステム

- [ ] **スキルカテゴリ**
  - 攻撃系（ダンジョンで使用）
  - 表情・顔文字表示
  - パーティー系（クラッカー・紙吹雪エフェクト）
  - アイテム系（お菓子をばら撒く）
  - 移動系（ランダムテレポート）
  - サポート系（回復・バフ）
  - 変身系（一時的に外見変化）

- [ ] **スキル例**
  - 火炎ブレス（攻撃）
  - 顔文字ポップアップ（表情）
  - パーティークラッカー（パーティー）
  - キャンディドロップ（アイテム）
  - ランダムワープ（移動）
  - ヒールオーラ（サポート）
  - プリズムシフト（変身）

## 🏠 部屋・飾り付け

- [ ] グリッドベース（16x12マス）の部屋
- [ ] 壁紙・床材の変更
- [ ] 家具の購入・配置・回転
- [ ] 家具にボーナス効果あり（例: 高級ベッド→睡眠回復+20%）
- [ ] 部屋の「快適度スコア」がペットの幸福度に影響

## 🌍 MMO・共有空間

- [ ] **公園・広場**
  - 他プレイヤーのペットとリアルタイムで集まれる
  - Socket.io で位置同期

- [ ] **ダンジョン**
  - ソロ・協力探索
  - 敵との戦闘（スキル使用）
  - 戦闘方式は要検討（リアルタイム or ターン制）

- [ ] **宝くじ店**
  - テレポートで移動して到着
  - サーバー側で抽選処理

- [ ] **マーケット**
  - プレイヤー間でドロップ品を売買
  - 掲示板型（非同期）
