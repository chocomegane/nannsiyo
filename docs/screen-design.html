2. コンポーネント一覧

コンポーネントファイル種別役割
Room components/Room.tsx UI 全体コンテナ。背景描画・ドロップループ起動・各子コンポーネントを配置
Pet components/Pet.tsx UI ペットの絵文字表示。Framer Motion で浮遊アニメーション（y ±12px / 2秒ループ）
DroppedItem components/DroppedItem.tsx UI 床に絶対位置で表示。クリックで回収。出現時 scale 0→1 アニメーション
MoneyDisplay components/MoneyDisplay.tsx UI 所持金 + 「全部売る」ボタン。インベントリ件数バッジ表示
petStore store/petStore.ts Store ペット状態（種族・レベル・ステータス）を管理
playerStore store/playerStore.ts Store お金・インベントリ・床上アイテムを管理。addDroppedItem / collectItem / sellAll
dropSystem systems/dropSystem.ts System 30〜60秒ランダムタイマーでアイテム生成。位置は床エリア内でランダム
